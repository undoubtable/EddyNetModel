# Baseline Reproduction Logs

This directory contains the final training logs for the SCSE-Eddy baseline reproduction experiments.

All baseline models use the same SCSE-Eddy data split, training configuration, loss, and evaluation protocol used in this repository.

The aggregated final metrics are available in `../../results/baseline_metrics.md`.

Original checkpoints, prediction arrays, probability maps, and the full `runs/` directory are intentionally not included in GitHub.
